  1. Parts & Tools Checklist
    • Parts: PCB, case, plate, switches, keycaps, stabilizers, optional foam, switch films, springs, lube
    • Tools: soldering iron (if needed), solder, switch puller, keycap puller, tweezers, screwdrivers, lubing tools, multimeter
  2. Prep & Workspace
    • Clear a clean, flat workspace
    • Unbox and verify layout & part compatibility
  3. Install Stabilizers
    • Clip and lube stabilizers
    • Install stabilizers onto the PCB or plate
  4. Plate + Switch Installation
    • Hot-swap PCB: Insert switches into plate → press into PCB sockets
    • Solder PCB: Insert switches → solder pins → check for clean joints
  5. Optional Switch Mods
    • Lube switches and springs
    • Add switch films
    • Swap springs if desired
  6. Assemble the Case
    • Place PCB + plate assembly into case
    • Add foam if desired
    • Screw case together
  7. Add Keycaps & Check Fit
    • Install keycaps according to layout
    • Ensure stabilizers feel smooth and keys align
  8. Firmware & Testing
    • Connect keyboard and test keys
    • Use VIA/QMK to flash firmware if needed
    • Verify every key registers correctly
  9. Sound & Aesthetic Tuning
    • Add foam mods, lube stabilizers, adjust switches
    • Customize keycaps, lighting, and cable
  10. Troubleshooting
    • Non-working key: reseat switch or inspect solder joint
    • Wobbly stabilizer: re-lube or switch to screw-in
    • Firmware errors: follow PCB bootloader instructions
